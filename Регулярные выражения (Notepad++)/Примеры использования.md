  
Регулярные выражения для поиска и замены:  
  
- Пустая строка через 2 абзаца:  
`\r\n\r\n`  
  
- Числа в формате {123} любой длины  
`\{\d+\}`  
  
- Строка, содержащая одну звездочку  
`\*\r\n`  
  
- Строка заканчивается на число любой длины  
Найти: `\d+\r\n`  
Заменить на: `\r\n`  
  
- Удалить подстроку, начинающуюся с символа нижнего подчеркивания и любого числа до конца строки  
Пример: F_AV_BRK_TOF_2;01.05.2023 0:00;173.91;0  
Выделит: `_2;01.05.2023 0:00;173.91;0`  
`_(\d+)(.+?)$`  
  
- Целое и вещественное число с точкой  
`(\d+(\.\d+)?)|(\d+)`  
  
- Все пробелы перед началом строки  
`\r\n\s+`  
  
- Любое количество пробелов  
`^\s*$`  
  
- Строки, где нет [operation], [reconcile], [dbo]  
`^(?!.*(\[operation\]|\[reconcile\]|\[dbo\])).+$`  
Заготовка: `^(?!.*()).+$`  
  
- Строка содержит фигурную скобку ( } ), но без запятой-разделителя. Использовал для поиска пропусков разделителей блоков JSON файлов  
`^}(?!.*,)`  
  
- Символ табуляции заменить на символ абзаца  
Найти: `\t`  
Заменить на: `\r\n`  
  
- Любая строка до символа =, заканчивающегося концом строки  
`^(.+?) = \r\n`  
  
- Любой текст между /* и */  
`/\*(.+?)\*/`  
  
- Подстрока Field_ID в строке  
`^(.+?)Field_ID(.+?)\r\n`  
  
- Заменить формат времени 0:00 на 00:00:00  
Найти: `(\b\d):(\d{2}\b)`  
Заменить на: `0$1:$2:00`  
  
- Заменить формат даты и времени 0000-00-00 00:00:00 на 00.00.0000 00:00 (с потерей секунд)  
Найти: `(\d{4})-(\d{2})-(\d{2}) (\d{2}):(\d{2}):(\d{2})`  
Заменить на: `$3.$2.$1 $4:$5`  
  
- Поменять местами в строке "_Month_NiQty" подстроку Month с любой последовательностью символов до конца  
Найти: `(_Month)(_\w+)`  
Заменить на: `\2\1`  
  
- Удалить кавычки возле четырехзначного числа  
Найти: `'(\d{4})'`  
Заменить на: `$1`  

- В выгрузке каталога удались весь текст, кроме расширений:  
Найти: .*(\.\w{2,7})$  
Заменить на: \1

- В тексте даты к номеру дня до 10 добавить цифру 0  
Было: 1 января 2024  
Стало: 01 января 2024  
Найти: ^(\d{1})  
Заменить на: 0\1

- Добавить табуляцию перед именем и расширением файла  

  Пример:  
  Исходный текст: F\Папка 1\Папка 2\Файл.pdf;31.14 Мб  
  Ожидаемый результат: F\Папка 1\Папка 2\ Файл .pdf;31.14 Мб  

  Найти: ^(.*\\)([^\\]+)(\.\w+);(.+)$  
  Заменить на: $1\t$2\t$3;\t$4
