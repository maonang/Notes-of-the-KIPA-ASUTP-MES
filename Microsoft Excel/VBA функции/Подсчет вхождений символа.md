# _VBA Функция_: Подсчет вхождений символа

```vba
Function CharCount(s_symb As String, s_str As String) As Long
    Dim i As Long
    Dim l_cnt As Long
    
    l_cnt = 0
    For i = 1 To Len(s_str)
        If s_symb = Mid(s_str, i, 1) Then
            l_cnt = l_cnt + 1
        End If
    Next i
    
    CharCount = l_cnt
End Function
```

## Описание  
Предназначена для подсчета количества вхождений определенного символа или короткой подстроки внутри заданной строки. Функция чувствительна к регистру.

## Синтаксис
```vba
Function CharCount(s_symb As String, s_str As String) As Long
```

**Параметры**:
| Параметр | Тип данных | Описание |
| --- | --- | --- |
| s_symb | String | Искомый символ или короткая строка, количество вхождений которой нужно посчитать |
| s_str | String | Исходная строка, в которой производится подсчет |

**Результат**:
- Общее количество вхождений s_symb в s_str.

## Сравнение
| Метод | Формула Excel | Описание |
| --- | --- | --- |
| VBA-функция | =CharCount("и"; A1) | Простой вызов функции с двумя аргументами |
| Формула | =ДЛСТР(A1) - ДЛСТР(ПОДСТАВИТЬ(A1; "и"; "")) | Вычисляет разницу между общей длиной строки и ее длиной после удаления искомого символа |
