# Горячие клавиши
- `F2` - переключает режим работы стрелок: перемещают курсор либо по символам формулы, либо по ячейкам листа;
- `F4` - при работе с формулами быстро добавляет знак доллара, фиксируя строки или столбцы при копировании;
- `Ctrl + ;` - вставляет текущую дату, а Ctrl + Shift + ; — текущее время;
- `Ctrl + стрелка` - позволяет мгновенно перейти к краю таблицы;
- `Alt + =` - моментально суммирует выделенные ячейки;
- `Ctrl + F3` - даёт возможность присвоить массиву уникальное имя. Вместо =СУММ(A1:A100) можно писать, например, =СУММ(Продажи).
- `Ctrl + T` - превращает набор ячеек в «умную» таблицу с фильтрами, форматированием и красивыми ссылками;
- `Ctrl + E` - автоматически распознаёт шаблоны. Например, если в столбце email ivan@example.com нужно выделить имя, достаточно вручную ввести «ivan» для первой строки и применить Ctrl + E — Excel заполнит остальные строки по шаблону.
