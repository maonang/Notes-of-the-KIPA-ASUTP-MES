# Поиск неиспользуемых параметров
  
## Описание  
Инструмент для анализа программ на языке Structured Text (ST) в MasterSCADA 4D с целью поиска неиспользуемых параметров и оптимизации связей.  
  
## Назначение  
- Выявление неиспользуемых параметров в ST-программах  
- Уменьшение количества избыточных связей в проекте  
- Оптимизация кода для повышения производительности  
  
## Результаты анализа  
- **Параметры с 0 повторений** - рекомендованы к удалению (кроме элементов перечислений - EnO)  
- **Параметры с 1+ повторений** - активно используются в программе  
  
## Инструкция по использованию  
  
1. **MasterSCADA 4D**:   
   - Откройте вкладку "Легенда"  
   - Скопируйте скомпилированный ST-код  
  
2. **Excel**:  
   - Нажмите кнопку **"Вставить и обработать код программы на языке ST"**  
   - Дождитесь завершения обработки (автоматический анализ)  
  
3. **Анализ результатов**:  
   - Просмотрите список параметров в соответствующих столбцах  
   - Определите параметры с нулевым использованием  
  
4. **Принятие решения**:  
   - Удалите неиспользуемые параметры из соответствующих разделов программы в MasterSCADA  
   - **Внимание**: Не удаляйте элементы перечислений (EnO), даже если они не используются  
  
## Поддерживаемые разделы переменных  
- VAR_INPUT - входные параметры  
- VAR_IN_OUT - входные параметры с возможностью изменения  
- VAR_OUTPUT - выходные параметры  
- Основной код программы  
  
## Особенности  
- Автоматическое удаление лишних пробелов при вставке кода  
- Подсчет использования переменных во всех разделах программы  
- Удобное визуальное представление результатов
