Функции программы:
- Формирует MVEL-выражение в отформатированном виде. Результат сохраняет в текстовый файл output_format.txt;
- Формирует обобщенную статистику по обработанному MVEL-выражению. Результат сохраняет в текстовый файл output_info.txt;
- Работа с записью из выгрузки ОМ вида: {"tagId":"220.1V.F_C3H6N6","variables":[{"type":"propertyId","alias":"A","value":"Модель тестовой передачи данных в SAP\\МЕЛАМИН ЦЕХА N2-А НВА\\WI-15205|Текущая выработка с начала суток","isTrigger":false}],"expression":"Fn.max($A, '*-3h','*', 0)","isFlowCalc":true,"triggerType":"Periodic","calculationType":"mvelExpression","offsetInSeconds":75900,"periodInSeconds":86400};
- Работа с записью MVEL-выражения без лишних данных вида: Fn.max($A, '*-3h','*', 0)
- Удаляет лишние пробелы, символы табуляции, переносы строк;
- Символ экспоненты переводит в верхний регистр;
- Добавляет недостающие пробелы (не обрабатывается оператор минус);
- Определяет четность количества скобок по каждому виду: (, [, {;
- Определяет четность количества кавычек по каждому виду: ", ';
- Исправляет разделитель вещественных чисел с символа "," на ".";
- Определяет перечень явно объявленных переменных (локальных) и группирует по типам данных;
- Определяет перечень прочих переменных (глобальных);
- Определяет переменные и выражения, которые потенциально будут переданы использованы в качестве результата выполнения всего MVEL-выражения. Признаком является отсутствие оператора присваивания;
- Вывод содержимого txt файлов в консоль при помощи команд +i и +f;
- Открытие файлов txt при помощи notepad.exe, используя команду +n;
- Открытие файла input.txt при помощи notepad.exe для добавления большого MVEL-выражения, используя команду +ni;

- Закрытие всех процессов notepad.exe, используя команду -n. Проверено на Windows 10;


Список команд:
	0 - Выход из программы;
	1 - Отформатировать MVEL-выражение: input.txt -> output_info.txt и output_format.txt;
	2 - Преобразовать исправленное отформатированное MVEL-выражение в одну строку: output_format.txt -> output_info.txt;
	+n - Открыть в блокноте файлы output_format.txt и output_info.txt;
	+ni - Открыть в блокноте файл input.txt;
	-n - Завершить все процессы notepad.exe. Проверено на Windows 10;
	+i - Отобразить содержимое файла output_info.txt в консоли;
	+f - Отобразить содержимое файла output_format.txt в консоли;
	help - Отобразить список команд.
