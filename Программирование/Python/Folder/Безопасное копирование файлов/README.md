# Безопасное копирование файлов

## Описание
Предназначена для безопасного копирования файлов между директориями в операционной системе Windows с расширенным логированием в формате CSV и интерактивным управлением конфликтами.  
Подходит для переноса больших массивов данных, анализа ошибок доступа и диагностики проблемных файлов.  

## Назначение
- Копирование файлов из одной директории в другую с сохранением структуры.
- Ведение детализированных CSV-логов копирования.
- Диагностика ошибок доступа, длинных путей, проблемных владельцев.
- Опциональная интерактивная обработка конфликтов файлов.

# Возможности
- Интерактивный CLI
  - запрос путей источника и назначения;
  - выбор режима обработки существующих файлов: заменить / пропустить / всегда заменять / всегда пропускать.
- Расширенное логирование в CSV
  - info.csv — успешно скопированные файлы;
  - error.csv — общие ошибки копирования;
  - error_owner.csv — ошибки доступа и владельцев;
  - error_length.csv — ошибки слишком длинных путей;
  - skipped.csv — пропущенные пользователем файлы;
  - owner_changed.csv — случаи изменения владельца (если реализовано в процессе выполнения программы).
- Поддержка больших директорий
  - подсчёт всех файлов в дереве;
  - индикатор прогресса через tqdm (если доступен).
- Работа в Windows
  - корректное извлечение владельцев файлов через win32security;
  - автоматическое создание выходных директорий;
  - обработка специфичных для Windows ошибок путей.

# Принцип работы
- При запуске программа запрашивает:
  - путь к директории-источнику;
  - путь к директории-назначению;
  - режим обработки конфликтов (индивидуальные запросы или автоматические действия).
- Программа рекурсивно собирает список всех файлов в исходной директории.
- Для каждого файла:
  - формируется путь для копирования;
  - при конфликте выполняется выбранная пользователем стратегия;
  - файл копируется с сохранением метаданных;
  - создаётся запись в соответствующем CSV-логе.
- По завершении пользователю выводится статистика, а в каталоге logs_csv/ формируются таблицы с детальной информацией о ходе процесса.
