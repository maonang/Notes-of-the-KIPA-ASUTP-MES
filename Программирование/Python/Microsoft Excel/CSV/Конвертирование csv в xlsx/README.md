# Конвертирование csv в xlsx

## Описание
Предназначена для автоматической пакетной конвертации CSV файлов в формат XLSX с автоматическим определением параметров файлов и поддержкой файлов любого размера.  

## Назначение
- Массовая конвертация CSV файлов в Excel формат
- Обработка больших файлов, превышающих лимиты Excel
- Автоматическое определение кодировки и разделителей
- Удобная пакетная обработка всех CSV файлов в директории

## Возможности
- Автоматически находит и обрабатывает все CSV файлы в текущей директории
- Обрабатывает файлы любого размера с разбивкой на несколько листов Excel
- Автоматически определяет кодировку файла (UTF-8, UTF-8-BOM, CP1251)
- Распознает различные разделители (`,`, `;`, `\t`, `|`)
- Автоматически применяет фильтры ко всем столбцам в Excel
- Информативные сообщения о ходе выполнения и статистика
- Визуализация процесса обработки для больших файлов
