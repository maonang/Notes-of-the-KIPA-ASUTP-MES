# Объединение файлов csv

## Описание
Предназначена для автоматического объединения CSV-файлов с одинаковой структурой заголовков. Инструмент анализирует набор CSV-файлов в директории в с программой, группирует их по совпадению полей и предоставляет пользователю возможность выборочного объединения.  

## Назначение
- Автоматическое обнаружение CSV-файлов с идентичной структурой данных
- Безопасное объединение данных без потери информации
- Визуализация процесса обработки с прогресс-баром
- Поддержка различных кодировок и разделителей

## Возможности
- Автоматическое определение кодировки файлов
- Определение разделителей - поддержка запятых, точек с запятой и табуляции как разделителей
- Визуальный прогресс - отображение прогресса чтения каждого файла через tqdm
- Автоматическая группировка файлов по хэшу заголовков
- Запрос подтверждения перед объединением каждой группы
- Отображение количества записей в исходных и результирующих файлах

## Принцип работы
- Программа ищет все CSV-файлы в текущей папке
- Анализ структуры - для каждого файла определяется кодировка, разделитель и заголовок
- Группировка - файлы группируются по хэшу заголовков методом SHA256
- Пользователь подтверждает объединение для каждой группы
- Файлы одной группы последовательно записываются в результирующий CSV
- Формируется итоговый файл с сохранением исходной структуры данных
