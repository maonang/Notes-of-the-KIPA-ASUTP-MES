**Описание**: Скрипт добавляет порядковые номера скобок в переданную на вход программы строку. --  
  
**Источник идеи**: Цветовое разделение скобок в редакторе формул Excel.  
  
**Особенности**:  
- Если в строке отсутствует хотя бы одна скобка, то будет запрошен повтор на ввод строки.  
- Если в строке больше 100 символов, она разбивается на части.  
- Пары скобок имеют одинаковый порядковый номер.  
- Предполагается, что порядок скобок не нарушен.  
- В исходную строку для отображения добавляются пробелы в определенных местах, чтобы номера скобок визуально не находились вплотную.
