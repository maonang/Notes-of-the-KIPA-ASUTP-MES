# JSON-структура в формулу Excel

## Описание
Предназначена для преобразования JSON-структур в строки формул, готовые для вставки в ячейки Microsoft Excel.  

## Назначение
- Преобразование JSON-строк в Excel-формулы для удобного отображения
- Автоматическое экранирование специальных символов для корректного отображения в Excel
- Обработка длинных текстовых значений с учетом ограничений Excel

## Возможности
- Две версии реализации:
  - Базовая версия - простое преобразование JSON-строки в формулу
  - Параметризованная версия - автоматическая обработка JSON-структуры с разбивкой длинных строк
- Автоматическое экранирование кавычек и специальных символов
- Проверка длины строк с предупреждениями о превышении лимитов Excel
- Поддержка всех типов данных JSON: объекты, массивы, строки, числа, булевы значения
- Интерактивный режим для удобного ввода данных

## Принцип работы
- Базовая версия:
  - Очищает JSON-строку от лишних пробелов и переносов
  - Экранирует двойные кавычки для корректного отображения в Excel
  - Проверяет общую длину строки и выдает предупреждение при превышении 255 символов
  - Форматирует результат как Excel-формулу
- Параметризованная версия:
  - Парсит JSON-строку в Python-объект
  - Рекурсивно обрабатывает все элементы структуры
  - Автоматически разбивает длинные строки (>254 символов) на части с использованием оператора сцепления (&)
  - Корректно преобразует булевы значения и другие типы данных
  - Форматирует итоговую строку как формулу Excel

Обе версии позволяют легко вставлять сложные JSON-структуры в ячейки Excel без ручного форматирования и экранирования.

## Пример: 
Было: 
```json
{"filter":[{"value":"Repeatability","fieldid":"34e62f8a-dac1-435b-8848-6178c7e1af4d"}],"fieldid":"34e62f8a-dac1-435b-8848-6178c7e1af4d","directoryid":"fe80d44a-7d01-4187-8ee4-4a3107874e34"}
```

Стало:
```vbnet
="{""filter"": [{""value"": """ & "Repeatability" & """, ""fieldid"": """ & "34e62f8a-dac1-435b-8848-6178c7e1af4d" & """}], ""fieldid"": """ & "34e62f8a-dac1-435b-8848-6178c7e1af4d" & """, ""directoryid"": """ & "fe80d44a-7d01-4187-8ee4-4a3107874e34" & """}"
```
