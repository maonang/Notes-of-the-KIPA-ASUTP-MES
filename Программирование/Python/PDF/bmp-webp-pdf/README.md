Сохранение изображений страниц книги из bmp в webp, а следом формирование из них pdf в формате A4 300 dpi с качествами 100, 90, 80, ...20, 10, 9,8, ... 2, 1% для выбора подходящего соотношения качества страниц и размера выходного файла.  

Файл скрипта должен находится в каталоге с изображениями. В процессе выполнения будет создана папка result, куда будут сохранены webp и pdf (19 файлов)  

Цель написания скрипта: создание файлов книг из скриншотов страниц.
