# Анализ градуировочной таблицы

## Описание
Предназначена для автоматического анализа градуировочных таблиц резервуаров и вычисления высокоточного коэффициента пересчета уровня жидкости в объем.  

## Назначение
- Автоматическое определение шага уровня из данных градуировочной таблицы
- Вычисление оптимального коэффициента пересчета с точностью до 20 знаков после запятой
- Анализ расхождений между фактическими и расчетными значениями объема
- Обработка данных из Excel-файлов без необходимости ручной настройки

# Возможности
- Автоматическое обнаружение xlsx-файлов в текущем каталоге
- Пользователь выбирает поля для уровня и объема из доступных в файле xlsx
- Автоматическое вычисление шага уровня с проверкой стабильности
- **Робастные алгоритмы** - устойчивые методы вычисления коэффициента, защищенные от выбросов и шума в данных
- Оптимизация коэффициента с постепенным увеличением точности до 20 знаков
- Использование Decimal для исключения ошибок округления

# Принцип работы
- Загрузка данных - программа автоматически находит и загружает Excel-файл из текущей директории
- Выбор колонок - пользователь интерактивно выбирает колонки, соответствующие уровню (в мм) и объему (в литрах)
- Анализ шага - система определяет шаг уровня путем анализа разностей между соседними значениями
- Вычисление коэффициента - используется комбинированный подход:
  - Линейная регрессия через метод наименьших квадратов
  - Медианный анализ разностей с фильтрацией выбросов
- Оптимизация точности - поэтапное уточнение коэффициента с увеличением точности от 1 до 20 знаков
- Вывод результатов - представление коэффициента в двух форматах: с точностью 15 и 20 знаков после запятой
