Начало файла:  
_Пример_:
```python
#!/usr/bin/env python3
# -*- coding: utf-8 -*-
```
_Описание_:
1. shebang (шейбанг)
```python
#!/usr/bin/env python3
```
- Строка нужна, если запускать файл через командную строку `./script.py`
- Говорит системе "запусти этот файл интерпретатором Python 3"
2. Кодировка файла UTF-8
```python
# -*- coding: utf-8 -*-
```

- Разделение блоков кода на секции
```python
# ------------------------------------------------------------------------------
# Текст
# ------------------------------------------------------------------------------
```

- Выполнение функции `main()` при запуске скрипта напрямую
```python
def main():
    ...

if __name__ == "__main__":
    main()
```
