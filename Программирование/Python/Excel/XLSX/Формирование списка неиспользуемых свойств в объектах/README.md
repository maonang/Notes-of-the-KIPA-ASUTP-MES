Формирование списка неиспользуемых свойств в объектах.  

Признаки для удаления записи:  
1) Значение поля Type не равно Property или PropertyConfiguration;  
2) Значение поля PropertyType равно Instance;  
3) Свойство не встречается в поле Configuration.  

Для работы требуется выгрузка из объектной модели.
