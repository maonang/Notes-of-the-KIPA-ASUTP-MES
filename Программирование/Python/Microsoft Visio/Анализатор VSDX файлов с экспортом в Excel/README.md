# Анализатор VSDX файлов с экспортом в Excel
  
## Описание  
Предназначена для автоматического анализа файлов Microsoft Visio (.vsdx), извлечения информации о всех элементах диаграмм и сохранения результатов в структурированном Excel файле с форматированием.

## Назначение  
- Анализ структуры Visio диаграмм
- Извлечение свойств фигур, текста и форматирования
- Экспорт данных в удобный для анализа формат Excel
- Автоматическая обработка нескольких файлов в пакетном режиме

# Возможности  
- Пакетная обработка всех .vsdx файлов в директории
- Рекурсивный обход всех фигур и вложенных элементов на всех страницах документа
- Извлечение полной информации о фигурах: координаты, размеры, текст, цвета, шрифты, гиперссылки
- Специальная обработка соединителей - определение типа, точек соединения, связанных фигур
- Очистка и нормализация текста с обработкой проблем кодировки
- Автоматическое форматирование Excel:
  - Цветовая заливка ячеек согласно цветам фигур
  - Автоподбор ширины колонок
  - Выделение заголовков
  - Адаптивный цвет текста для темных фонов
- Обработка ошибок с детальным логированием процесса
