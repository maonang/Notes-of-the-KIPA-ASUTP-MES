## Примеры  
 Масса   
- Действительный массовый расход, т/ч - Qm  
- Измеренный массовый расход, кг/ч, т/ч - Qme  
- Массовый расход среды при рабочих условиях, кг/ч - Qm  
- Расчетное значение массового расхода для расходомера, кг/ч, т/ч - Qce  
  
 Объем   
- Действительный объемный расход, м3/ч - Qv  
- Измеренное значение объемного расхода при рабочих условиях, м3/ч - Qv  
- Объемный расход среды при стандартных условиях, м3/ч - Dz  
- Объемный расход, м3/ч - Qve  
- Расчетное значение объемного расхода для расходомера, м3/ч - Qce  
  
 Давление   
- Абсолютное давление среды, кгс/см2 - P  
- Барометрическое давление, мм.рт.ст - Patm  
- Верхний предел измерения датчика перепада давления, кгс/м2 - dPhi  
- Давление насыщения, кгс/см2 - Ps  
- Избыточное давление, кгс/см2 = Pme  
- Измеренное значение избыточного давления, кгс/см2 - Pme  
- Измеренное значение перепада давления, кгс/см2 - dPme  
- Перепад давления на СУ, кгс/см2 - dPme  
- Приведенное абсолютное давление, кгс/см2 - Pp  
- Расчетное значение давления для расходомера, кгс/см2 - Pr  
- Расчетное значение перепада давления для расходомера, кгс/см2 - dPce  
  
 Плотность   
- Плотность при рабочих условиях, кг/м3 - Ro  
- Плотность при стандартных условиях, кг/м3 - Rostme  
- Расчетное значение плотности среды для расходомера, кг/м2 - Ror  
  
 Температура   
- Абсолютная температура среды, °C - T  
- Измеренное значение температуры, °C - Tme  
- Приведенная температура, °C - Tp  
- Расчетное значение температуры для расходомера, °C - Tr  
- Температура измеряемой среды, °C - Tme  
  
 Концентрация   
- Содержание азота, моль - Xa  
- Содержание углекислого газа, моль - Xco2  
  
 Коэффициенты   
- Коэффициент расширения среды - Kexp  
- Коэффициент сжимаемости - Kg  
- Коэффициент скорости входа - E  
- Поправочный коэффициент на тепловое расширение материала ИТ - Ktit  
- Поправочный коэффициент на тепловое расширение материала СУ - Ktsu  
- Температурный коэффициент линейного расширения материала измерительного трубопровода (ИТ), 1/°C - Git  
- Температурный коэффициент линейного расширения материала СУ, 1/°C - Gsu  
  
 Прочее   
- Внутренний диаметр ИТ перед СУ при стандартных условиях, мм - Dit20  
- Внутренний диаметр ИТ при рабочих условиях, мм - Dit  
- Диаметр отверстия диафрагмы при рабочих условиях, мм - Dsu  
- Диаметр отверстия СУ при стандартных условиях, мм - Dsu20  
- Динамическая вязкость, Па*с - Mu  
- Код среды - KodSr  
- Концентрация аммиака, % - Hme  
- Марка стали ИТ - msIT  
- Марка стали СУ - msSU  
- Начальный радиус закругления входной кромки диафрагмы, мм - Rkrn  
- Относительный диаметр диафрагмы - Bt  
- Период поверки диафрагмы, год - Tau  
- Фактор сжимаемости при рабочих условиях - Z  
- Фактор сжимаемости при стандартных условиях - Zst  
- Эквивалентная шероховатость внутренней поверхности ИТ, мм - Rh  
- Энтальпия, кДж/кг - Ht
