### Баланс  
  
Обобщенная формула:  
<Сумма входящих потоков> = <Сумма выходящих потоков>  
  
### Общая формула:  
<Выходящие потоки> + <Остаток на конец> - <Входящие потоки> - <Остаток на начало> + <Нормированные потери> = <Дебаланс>,  
где <Входящие потоки> - это масса вошедших материалов в объект. Измеряемые данные с заданной погрешностью.  
Типы входящих потоков:  
- измеряемые данные (данные с измерительных приборов, ручного ввода, смежных систем). Могут быть избыточными и неизбыточными;  
- неизмеряемые данные (отсутствует информация о движении/запасе). Могут быть наблюдаемые и ненаблюдаемые;  
- фиксированные значения (данные коммерческого учета, финансовые документы, инвентаризация, нормированные потери). Погрешность равна 0, поэтому в условиях баланса участвует в качестве константы.  
<Выходящие потоки> - Масса вышедших материалов из объекта;  
<Остаток на начало> - Масса материалов внутри объекта на начало балансового периода;  
<Остаток на конец> - Масса материалов внутри объекта на конец балансового периода;  
  
### Объекты балансовой модели:  
- Точка входа = приемка/поступление сырья, энергоресурсов, и т.д.  
Является границей балансовой модели, из которой материал приходит (вход) в модель. Не является объектом, для которого необходимо составлять балансовое уравнение.  
- Точка выхода = отгрузка ГП, отходы производства, окружающая среда, и т.д.  
Является границей балансовой модели, в которую материал уходит (выход) из модели. Не является объектом, для которого необходимо составлять балансовое уравнение.  
- Балансовый объект без запасов = установки, коллектора, переделы и т.д.  
Балансовое уравнение: <Сумма входящих потоков> = <Сумма выходящих потоков> + <Потери>  
- Балансовый объект с запасами = емкости, резервуары, склады сырья и ГП, и т.д.  
Балансовое уравнение: <Входящие потоки> + <Остаток на начало> = <Выходящие потоки> + <Остаток на конец> + <Потери>  
- Измерительные средства, экспертная оценка, косвенный расчет.  
Является оценкой количества движения или запаса материала с заданной погрешностью;  
- Данные коммерческого учета, не подлежащие корректировке в процессе сведения баланса.  
Является неизменной величиной, т.к. имеет прямую привязку к финансовым обязательствам перед сторонними организациями.  
  
Баланс сводится только с учетом фактических данных и погрешности средств измерений. Плановые значения производства не учитываются.  
Избыточность данных позволяет рассчитать поток несколькими способами. Чем больше способов рассчитать поток (систем уравнений), тем выше точность сведения баланса. Избыточность достигается путем добавления измерительных приборов.  
  
### Сведение материального баланса:  
### На вход в балансовую модель поступает:  
- перечень балансовых объектов;  
- перечень граничных объектов;  
- связи объектов друг с другом (движения);  
- оценка (значения) движения, запасов, потерь;  
- погрешности значений;  
- метрологические и технологические ограничения;  
- корректировка исходных данных, потерь.  
### На выход из балансовой модели поступает:  
- согласованные значения всех движений, запасов, потерь;  
- аналитические показатели результатов потерь.  
  
### Цели сведения материальных балансов:  
- уточнение данных измерений = определить как отработало предприятие за производственный период;  
- обеспечение нулевого дебаланса;  
- идентификация ошибок измерений;  
- идентификация и минимизация потерь.  
  
### Виды балансовых моделей:  
- сквозная. Заключается в отражении всех балансовые объектов установки от поставки/добычи сырья (на входе) до выпуска продукции (на выходе);  
- по среде. Заключается в выделении из сквозной балансовой модели отдельных потоков по средам, например, газу, сыпучим продуктам. Минус - отсутствие понимания связей между потоками с разным сырьем. Выход - строгая последовательность сведения баланса;  
- по участкам. Заключается в выделении из сквозной балансовой модели отдельных потоков по установкам, относящихся к конкретному участку/цеху. Минус - отсутствие понимания связей между потоками с разным сырьем. Выход - строгая последовательность сведения баланса;  
  
### Потери:  
- По каждому балансовому объекту должен быть определен перечень возможных потерь и их классификация;  
- Необходимо определить подходы к конфигурированию неизмеряемых потерь;  
- Наличие неизмеряемого граничного потока в модели - это возможность для системы сведения баланса списать весь дебаланс объекта. Вариант рещения: изначально неизмеряемые потери равны 0. Значение правится по итогам анализа дебалансов и заседания балансовой комиссии.  
  
### Классификация потерь по месту возникновения:  
- технологические = от подачи сырья в технологический процесс до выдачи готовой продукции на склады. Пример: утечка, неплотности (например, соединительных муфт), дренаж, сжигание на факеле, отходы производства;  
- товарные = приемка сырья от поставщиков и его хранение, хранение готовой продукции и его отгрузка потребителям. Пример: просыпь, отбор проб, испарения, пролив.  
  
### Классификация потерь по возможности управления:  
- управляемые = дренаж, сжигание на факеле, свечах, отходы производства;  
- неуправляемые = испарения, неплотности оборудования, утечки, проливы, аварийные потери, отбор проб.  
  
### Классификация потерь по возможности оценки значений:  
- измеряемые - наличие средств измерения или расчетной оценки;  
- неизмеряемые - отсутствие информации о количестве. Пример: утечка, пролив, просыпь.  
  
**Согласованные данные - данные, обеспечивающие нулевой дебаланс.**  
**Погрешность - относительная погрешность измеренных данных;**  
**Метрологические границы - ограничения в рамках погрешности средства измерения. В этих границах система должна свести баланс;**  
**Технологические границы - физические ограничения, например, трубы, конвейера, емкости по параметрам: минимальная/максимальная пропускная способность/вместимость.**  
**Потоки движения = расходомер, конвейер, весы, документы отгрузки, экспертная оценка, плановые расходные нормы, косвенный расчет;**  
**Потоки запаса = уровнемеры, инвентаризация, экспертная оценка, маркшейдерский замер, запасы на колесах, косвенный расчет;**  
  
**Расчет выполняется методом наименьших квадратов**
