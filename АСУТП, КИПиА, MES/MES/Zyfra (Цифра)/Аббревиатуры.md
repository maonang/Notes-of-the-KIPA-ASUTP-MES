## Аббревиатуры

### Продукты Zyfra
- **Z-EFAA** (Zyfra Energy Forecast, Accounting and Analytics) - Прикладное приложение для минимизации потребления технической электроэнергии
- **Z-SPM** (Zyfra Smart Plant Monitoring) - Прикладное приложение для интеллектуального управления производством
- **ZIAK** (Zyfra Industrial Automation Kit) - Набор инструментов для автоматизации промышленности
- **ZIF** (Zyfra Industrial Framework) - Индустриальный фреймворк для разработки решений
- **ZIIoT** (Zyfra Industrial Internet of Things Platform) - Платформа для промышленного интернета вещей
- **ZQA** (Zyfra Quality Assurance) - Источник данных и инструмент для контроля качества
- **ZQL** (Zyfra Quality Lab) - Прикладное приложение для управления качеством

### AI-решения
- **DA** (Digital Adviser / Avatar) - Цифровые советчики / аватары
- **DBRPA** (Digital Bots Robotic Process Automation) - Цифровые боты RPA
- **DDA** (Digital Data Analyses) - Цифровой анализ данных
- **DO** (Digital Optimizer) - Цифровые оптимизаторы
- **DP** (Digital Forecast) - Цифровой прогноз
- **DPP** (Digital Production Planning) - Цифровое планирование производства
- **DSM** (Digital Simulation Modeling) - Цифровое имитационное моделирование
- **DT** (Digital Twins) - Цифровые двойники
- **DV** (Digital Video) - Цифровое видео

### Система подготовки данных (СПД):
- **DAG** (от англ. Directed Acyclic Graphs) - однонаправленный граф с равноправными элементами. Схема не содержит циклы. В схеме могут быть параллельные пути, выходящие из одного узла.
