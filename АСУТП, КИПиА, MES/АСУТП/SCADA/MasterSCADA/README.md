# Горячие клавиши   
   
Когда выделен объект на мнемосхеме:   
- Shift + Left/Right/Up/Down - Изменить размер объекта   
- Shift + ЛКМ - выравнивание по горизонтали/вертикали при переносе объекта  
- Shift + ЛКМ (по невыделенному) - добавить к выделенным объектам   
- Ctrl + Left/Right/Up/Down - Изменить расположение объектов с шагом 1  
- Ctrl + ЛКМ (по невыделенному) - добавить к выделенным объектам  
- Ctrl + ЛКМ (по выделенному) - исключить из выделенных объектов  
- Ctrl + Выделение ЛКМ - выделение всех объектов, попадающих в область выделения  
- Перемещение выделенных объектов + Shift - выравнивание по оси или диагонали (зависит от направления движения мышки)  
   
Когда выбран элемент в дереве проекта:   
- Ctrl + Up - Перенести вверх по списку  
- Ctrl + Down - Перенести вниз по списку  
  
- Shift+Выделить ЛКМ область в окне - увеличить до масштаба выделенной области  
- Скролл - прокручивание окна вверх/вниз  
- Shift+Скролл - прокручивание окна влево/вправо  
- Ctrl+Скролл - масштабирование
